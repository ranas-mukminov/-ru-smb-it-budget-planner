# ru-smb-it-budget-planner

**Инструмент планирования и сравнения ИТ-бюджетов для малого и среднего бизнеса в РФ.**

Этот инструмент помогает оценить затраты на ИТ-инфраструктуру, сравнивая сценарии:
1. **On-Premise** (собственное оборудование).
2. **Colocation** (размещение в ЦОД).
3. **Российские облака** (Yandex Cloud, VK Cloud, Selectel и др. - абстрактные профили).

Инструмент учитывает требования законодательства РФ (152-ФЗ, КИИ) при распределении нагрузок.

## Быстрый старт

### Установка

```bash
git clone https://github.com/ranas-mukminov/ru-smb-it-budget-planner.git
cd ru-smb-it-budget-planner
pip install .
```

### Использование

1. Подготовьте файл описания инфраструктуры `infra.yaml` (см. `examples/infra.yaml`).
2. Подготовьте (или используйте стандартный) файл цен `pricing.yaml` (см. `examples/pricing.yaml`).
3. Запустите расчет:

```bash
ru-smb-planner plan --infra infra.yaml --pricing pricing.yaml
```

## Ограничения и дисклеймер

> [!IMPORTANT]
> Этот инструмент предоставляет **исключительно оценочные данные**.
> Результаты расчетов **не являются** финансовой офертой, юридической консультацией или гарантией будущих расходов.
> Реальные цены зависят от конкретных поставщиков, курса валют и условий договоров.
