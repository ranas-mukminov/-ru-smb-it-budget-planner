# ru-smb-it-budget-planner

**Инструмент планирования ИТ-бюджета для малого и среднего бизнеса в РФ.**

Этот инструмент позволяет описать вашу ИТ-инфраструктуру в формате YAML и автоматически рассчитать стоимость владения (TCO) для различных сценариев развертывания:
- **As Is**: Текущая конфигурация (On-Premise / Colo).
- **Minimal Cloud**: Миграция веб-нагрузок в российские облака.
- **Hybrid**: Гибридный сценарий с учетом требований 152-ФЗ и 187-ФЗ (КИИ).

## Для кого
Для собственников бизнеса, технических директоров и системных администраторов, которые хотят:
- Понять реальную стоимость своей ИТ-инфраструктуры.
- Сравнить затраты на "свое железо" vs "облака".
- Оценить экономию от миграции.

## Быстрый старт

### Установка

```bash
pip install .
```

### Использование

1. **Создайте примеры конфигурации:**

```bash
ru-smb-it-budget-planner init-sample
```

Это создаст файлы `infra.yaml` (описание инфраструктуры) и `pricing.yaml` (тарифы).

2. **Проверьте корректность конфигурации:**

```bash
ru-smb-it-budget-planner validate infra.yaml
```

3. **Рассчитайте бюджет:**

```bash
ru-smb-it-budget-planner plan infra.yaml pricing.yaml
```

Вы увидите таблицу со сравнением сценариев по стоимости (CAPEX/OPEX).

4. **Сгенерируйте отчет для руководства:**

```bash
ru-smb-it-budget-planner report infra.yaml pricing.yaml -o report.txt
```

## Ограничения и дисклеймер

> [!IMPORTANT]
> Это **оценочный инструмент**, а не юридическая или финансовая консультация.
> Реальные цены могут отличаться. Используйте актуальные тарифы ваших поставщиков в `pricing.yaml`.

Инструмент учитывает флаги `has_pd` (152-ФЗ) и `has_kii` (187-ФЗ) для рекомендаций по размещению, но **не гарантирует** автоматического соответствия законодательству.
